없음 현재 검토 대상으로 분류된 활동이 없습니다.

활동이 검토 대상으로 분류되면 여기에 사유가 통보되고 소명 제출 버튼이 표시됩니다. (예시: 관리자 화면 ⑤의 오세훈 건)

미팅 논의 포인트 — 점수 상세

  1. 산식 버전을 화면에 고정 표기 — "학기 시작 시 산식 고정, 학기 중 소급 변경 금지" 원칙(PRD)의 UI 표현.
  2. 근거 drill-down에 활동 출처 태그(course/external/personal)가 이미 붙어 있음 — v2 트랙 분리로 갈 때 데이터 구조 변경 없이 확장 가능.
  3. 누락 신고는 이의제기의 1차 통로. 신고→재수집→재산정 처리 기한(SLA)을 사업단과 합의 필요(Q8).
  4. 어뷰징 의심 활동은 자동 감점 없이 보류+통보+소명 구조. 판정은 위원회 몫(Q11).
